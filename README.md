# Project Name

Тестовое задания для digital-агенства Only

## Технологии

- React
- TypeScript
- Webpack
- Sass
- GSAP
- Swiper
- Yarn

## Установка

1. **Клонировать репозиторий:**

```bash
git clone https://github.com/Yanius27/only_task.git
cd only_task
```

2. **Установить зависимости:**

```bash
yarn install
# или
npm install
```

3. **Запустить проект:**

```bash
yarn start
# или
npm start
```
